# gap-secret-editor

> An electron-vue project

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# run all tests
npm test

#run unit tests once
npm run test:unit

#run unit tests in watch mode
npm run test:unit:watch

```
